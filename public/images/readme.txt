Cho thư mục images (hiện đang dùng dạng SVG):
- customer.svg: hình khách hàng cầm sách, xếp hàng.
- counter.svg: hình quầy mượn sách.
- floor.svg: nền sàn thư viện.
- librarian.svg: hình thủ thư.
- table_free.svg: hình bàn trống (bàn rảnh).
- table_busy.svg: hình bàn đang có người sử dụng (bàn bận).

Bạn có thể thay các file SVG demo này bằng hình thiết kế riêng (giữ nguyên tên file) để giao diện mô phỏng đẹp hơn.
